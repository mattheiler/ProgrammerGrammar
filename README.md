# Programmer Grammar

Project and items templates for:
- Command
- Identity Gateway
- Query
- Web Application

## Setup

### Add package source

```
dotnet nuget add source https://nuget.pkg.github.com/mattheiler/index.json
```

### Add package

```
dotnet add PROJECT package ProgrammerGrammar
```

## Usage

```
dotnet new command
dotnet new identitygateway
dotnet new query
dotnet new webapplication
```
